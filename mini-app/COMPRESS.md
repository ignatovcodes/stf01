# Сжатие фото блюд для Git

Чтобы уложиться в лимиты GitHub при пуше, сожмите изображения в папке `images/`.

## Как сжать (рекомендуется)

В корне проекта `mini-app` в терминале выполните:

```bash
npm install
npm run compress-images
```

Скрипт:
- изменит размер каждой картинки до максимум 800px по большей стороне (достаточно для карточек меню и retina);
- перезапишет PNG с максимальным сжатием (качество остаётся хорошим для веба).

Ожидаемый результат: было ~89 MB → станет ~15–25 MB. После этого коммит и пуш в Git должны проходить без ошибок.

## Если нет Node.js

1. Установите [Node.js](https://nodejs.org/) (LTS), затем снова выполните команды выше.
2. Либо используйте онлайн-сервис [Squoosh](https://squoosh.app): загрузите изображения пачкой, выберите формат PNG и уменьшите размер/качество, затем скачайте и замените файлы в `images/`.
