# Что сделать, чтобы бот в MAX работал

Бот должен жить в интернете по адресу с **https://**. Ниже — самый простой путь.

---

## Шаг 1. Зарегистрироваться в Timeweb Cloud

1. Зайди на сайт **https://timeweb.cloud**
2. Нажми **«Регистрация»** или **«Войти»**
3. Создай аккаунт (если его ещё нет)

---

## Шаг 2. Создать приложение (деплой)

1. В личном кабинете Timeweb найди раздел **«App Platform»** (или **«Приложения»** / **«Деплой приложений»**)
2. Нажми **«Создать приложение»**
3. Выбери **«Подключить репозиторий»** (GitHub, GitLab или другой Git)
   - Если кода ещё нет в Git — сначала залей папку **mini-app** в GitHub/GitLab (создай новый репозиторий и загрузи туда файлы из папки mini-app)
4. Укажи свой репозиторий и ветку
5. Если в репозитории проект лежит в папке **mini-app**, в настройках приложения укажи **«Корневая директория»** / **«Путь к приложению»**: `mini-app`
6. Среда: **Node.js**
7. Нажми **«Развернуть»** / **«Деплой»** и дождись, пока приложение поднимется

---

## Шаг 3. Узнать адрес приложения и добавить переменные

1. После деплоя Timeweb покажет **адрес приложения**, например:  
   `https://какой-то-адрес.timeweb.cloud`  
   Скопируй его полностью (без `/webhook` в конце).

2. В настройках этого приложения найди раздел **«Переменные окружения»** / **«Environment variables»**.

3. Добавь три переменные (кнопка «Добавить» / «Add»):

   | Имя | Значение |
   |-----|----------|
   | `MAX_BOT_TOKEN` | Токен бота. Взять: зайди на **business.max.ru** → Чат-боты → твой бот → Интеграция → **«Получить токен»**. Скопируй и вставь сюда. |
   | `WEBHOOK_SECRET` | Любой пароль, который придумаешь сам (например: `moy_parol_123`). Запомни его — понадобится в шаге 5. |
   | `MINI_APP_URL` | Тот самый адрес приложения из пункта 1, например: `https://какой-то-адрес.timeweb.cloud` |

4. Сохрани переменные и перезапусти приложение (если есть кнопка «Перезапустить»).

---

## Шаг 4. Проверить, что сайт открывается

Открой в браузере адрес из шага 3 (тот же, что в `MINI_APP_URL`).  
Должна открыться страница ресторана Стефания (меню, кнопки). Если открывается — всё ок, идём дальше.

---

## Шаг 5. Сказать MAX, куда слать события (подписать вебхук)

MAX должен знать адрес твоего бота. Это делается один раз — через запрос к API.

**Вариант А — через браузер (если есть расширение для отправки запросов):**  
Мало кто так делает, поэтому лучше вариант Б.

**Вариант Б — через командную строку (PowerShell на Windows):**

1. Открой **PowerShell** (поиск Windows → «PowerShell»).
2. Вставь одну команду (одной строкой), **предварительно замени**:
   - `ТВОЙ_ТОКЕН_БОТА` — тот же токен, что в переменной `MAX_BOT_TOKEN`;
   - `АДРЕС_ПРИЛОЖЕНИЯ` — адрес из шага 3 **с добавлением в конец** `/webhook`, например: `https://какой-то-адрес.timeweb.cloud/webhook`;
   - `ТВОЙ_WEBHOOK_SECRET` — тот же пароль, что в переменной `WEBHOOK_SECRET`.

Команда:

```powershell
Invoke-RestMethod -Uri "https://platform-api.max.ru/subscriptions" -Method POST -Headers @{ "Authorization" = "ТВОЙ_ТОКЕН_БОТА"; "Content-Type" = "application/json" } -Body '{"url": "АДРЕС_ПРИЛОЖЕНИЯ", "update_types": ["bot_started", "message_created"], "secret": "ТВОЙ_WEBHOOK_SECRET"}'
```

Пример (подставь свои данные):

- Токен: `abc123xyz`
- Адрес приложения: `https://stefania-12345.timeweb.cloud`
- Значит url для вебхука: `https://stefania-12345.timeweb.cloud/webhook`
- Секрет: `moy_parol_123`

Тогда команда будет такой:

```powershell
Invoke-RestMethod -Uri "https://platform-api.max.ru/subscriptions" -Method POST -Headers @{ "Authorization" = "abc123xyz"; "Content-Type" = "application/json" } -Body '{"url": "https://stefania-12345.timeweb.cloud/webhook", "update_types": ["bot_started", "message_created"], "secret": "moy_parol_123"}'
```

3. Нажми Enter. Ошибки в ответе не должно быть (если есть — напиши, разберём).

---

## Шаг 6. Проверить в MAX

1. Открой мессенджер **MAX**
2. Найди своего бота и нажми **«Старт»** (или напиши `/start`)
3. Бот должен прислать приветствие от ресторана Стефания и кнопку **«Открыть меню и сделать заказ»**
4. Нажми на кнопку — должно открыться твоё мини-приложение (тот же сайт, что открывался в шаге 4)

Если приветствие не приходит — проверь, что в шаге 5 подставлены правильные токен, адрес с `/webhook` и тот же секрет, что в переменных.
